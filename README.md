## **nest-sql-practice-2 DDB Backend Challenge**


## Description

Based on the DnDBeyond backend challenge that can be found here: https://github.com/DnDBeyond/back-end-developer-challenge

## Project setup

Install dependencies:

```bash
$ npm install
```

Run project: 
```bash
# watch mode
$ npm run start:dev

```
